Завдання 2. medium] Модифікувати TCP-сервер з п.1, щоб він працював постійно та обробляв запити від багатьох клієнтів послідовно.
TCP-сервер з попереднього завдання перероблено, щоб той працював в постійному режимі.
Запущено двох клієнтів. У консолі можна побачити, що сервер прийняв з'єднання лише з першого. По завершенню обробки повідомлення від першого клієнта, в консолі можна побачити, що сервер прийняв підключення від другого клієнта. Це означає, що сервер обслуговує клієнтів послідовно.
